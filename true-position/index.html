真位置度判定穴位置の合否 = 2×√(ΔX²+ΔY²)

使い方

  1. 位置度公差 φ（図面の公差記入枠の値）を入れます。
  2. 図面値 X・Y と 実測値 X・Y を入れると、ΔX・ΔY と位置度が出ます。
  3. 差（ΔX・ΔY）だけ分かっているときは、図面値を 0・0 にして実測値に差を入れてもOKです。
  4. 位置度＝2×√(ΔX²+ΔY²)。これが φ 以下なら合格。Ⓜ（最大実体）ならボーナス公差も加味します。

例 ── ずれ ΔX0.03・ΔY0.04

公差 φ0.2。図面値 (0,0) に対して実測が (0.03, 0.04) ずれていたら？

φ 0.2 ／ 図面値 X0 Y0 ／ 実測値 X0.03 Y0.04

読み方：位置度 = 2×√(0.03²+0.04²) = 0.100 ≤ 0.2 なので 合格。
