検査ミニポータル 一式(2026-07-12 時点)
====================================

GitHub Pages 用のフォルダ構成そのままです。
リポジトリ直下に kensa-tools/ の中身を置いてください。

・index.html … メニュー(検索・お気に入り付き)
・各フォルダ/index.html … 各ツール(全26種)

全ツールとも外部通信なし・端末内で計算のみ。
スマホでは URL を開いて「ホーム画面に追加」で使えます。

今回の追加(2026-07-12):
・TOOL-25 corner-dist  角－角 距離
    直方体の対角と、隅切り・切欠きなど凹み形状の全頂点間距離。
・TOOL-26 axis-tilt    軸の傾き
    検査棒の2点読みから角度・勾配・直角度。すき間ロッキングの有効性判定つき。
※ 別チャットで一時的に TOOL-22/23 と呼んでいましたが、本番の採番に合わせ
   TOOL-25/26 に確定しました(22=穴位置ビルダー, 23=カム変位線図, 24=直角三角形)。
