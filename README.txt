検査ミニポータル 一式(2026-07-14 時点・オフライン対応版)
====================================

GitHub Pages 用のフォルダ構成そのままです。
リポジトリ直下に kensa-tools/ の中身を置いてください。

・index.html … メニュー(検索・お気に入り付き)
・各フォルダ/index.html … 各ツール(全28種)
・sw.js / manifest.json / icon-*.png … オフライン(PWA)用

全ツールとも外部通信なし・端末内で計算のみ。

■ オフライン専用機としての使い方
1. 専用にするスマホで、一度だけオンラインでトップページを開く。
   → 「オフライン準備が完了しました」と出れば、全28ツールが端末に保存済み。
2. ブラウザメニューの「ホーム画面に追加」でアイコンを作成。
   → 全画面アプリとして起動し、以後は機内モードでも全ツールが動きます。
3. 手が汚れる作業前に機内モード/オフラインにしておけば通信は一切発生しません。

■ ツールを更新したとき（作者向け）
・sw.js の 1 行目付近 var CACHE = "kensa-tools-v1" の数字を v2, v3… と上げる。
・一式を再アップロードし、専用機で一度オンラインで開く。
  → 「新しいバージョンを取得しました」と出て、次回起動時に最新へ入れ替わります。
・番号を上げ忘れると、専用機は古いキャッシュのままになります(オフライン優先のため)。
