使い方

  1. 基準は A（ここが 0,0）。「点を追加」フォームで ①どこから測るか → ②どう置くか → ③数値 の順に入れて「この位置に追加」。
  2. 入力欄のラベルに基準名が入ります（例:「Bからの距離」）。入力中は図の基準点が青くなり、寸法線・角度がその場で描かれます。
  3. 置き方は3つ:「距離と角度で」「直角に振り出す」(基準の向きから左右90°)「等ピッチで並べる」(15°ずつ×3個 など一括)。
  4. 追加した点は上の一覧に「B: Aから 距離50・角度30°」のように残り、編集・削除できます。角度は真上0°・時計回り＋(度分秒可)。

例1 ── Aから2点を置いて距離を見る

Aから 距離50・角度0°で B、続けて 距離50・角度90°で C を追加。

読み方：B→C の距離 = 70.711（=50√2）。ΔX・ΔY も下段に出ます。

例2 ── 置いた点から直角に振り出す

Aから 距離40・角度30°で B。次に基準をBにして「直角に振り出す」を選び、距離20・右へ。「基準の向き」にはBの置き角30°が自動で入ります。

読み方：Cの絶対角は 30°+90°=120°。A→C の距離 = 44.721（=√(40²+20²)）。

例3 ── 等ピッチで並べる（15°ずつ×3個）

基準A・「等ピッチで並べる」で r40・15°ずつ・3個。B1=15°・B2=30°・B3=45° が一括で入ります（開始角は空ならピッチと同じ）。

読み方：B1→B3 の距離 = 20.706（=2×40×sin15°）。
